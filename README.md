# FitnessCenter
This project is a comprehensive gym management solution developed using ASP.NET Core MVC architecture as part of the Web Programming course. The system aims to digitize gym service processes, increase trainer-member interaction, and provide a personalized experience powered by artificial intelligence.

Project Key Features:

Appointment Management: Members can schedule online appointments based on trainers' availability and expertise.

Artificial Intelligence Integration: Users can receive AI-based exercise and diet program recommendations based on their physical characteristics.

Trainer and Gym Management: Detailed tracking of gym operating hours, service types, and trainer specialties.

Role-Based Authorization: Secure access control with Admin and Member panels.

Technological Infrastructure: .NET (Core) LTS, Entity Framework Core, LINQ, REST API, and Bootstrap 5
